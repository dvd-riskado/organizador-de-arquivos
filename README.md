Hi! Esse aqui é meu primeiro projeto realmente "complexo" em python!
O programa que você está vendo é um organizador de arquivos, inspirado em um separador de itens no minecraft.
O programa lê seus arquivos em Dowloads, e os move para as pastas "padrão" dependendo do tipo de arquivo:

-> Fluxo dos arquivos por estensão:
- Imagens (.png, .jpg, etc) -> /home/seu_user/Imagens
- Texto (.pdf, .docx, etc) -> /home/seu_user/Documentos
- Áudio (.mp3, .ogg, etc) -> /home/seu_user/Músicas
- Vídeo (.mp4) -> /home/seu_user/Vídeos

O programa foi especialmente pensado para Linux, mais especificamente, para distros baseadas em debian
